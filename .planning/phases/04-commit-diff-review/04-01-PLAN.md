  </behavior>
  <action>
    Create lua/commit_picker/git.lua:

    Use `io.popen()` to run git commands. Follow existing patterns from `lua/ai/fetch_models.lua` for process execution.

    `get_commit_list(base, head)`:
    - Run: `git log --format=%H%n%h%n%s%n%cr%n%d base..head`
    - Parse output in groups of 5 lines per commit
    - Return array of `{sha, short_sha, subject, date, refs}`
    - refs = branch/tag decorations, trimmed

    `get_unpushed()`:
    - Wrapper: `get_commit_list("origin/HEAD", "HEAD")`
    - Returns empty array if no unpushed commits (ahead=0)

    `get_ahead_behind()`:
    - Run: `git rev-list --left-right --count origin/HEAD...HEAD`
    - Parse stdout: first number = behind, second = ahead
    - Return `{ahead = N, behind = M}`

    Error handling:
    - If `io.popen` fails or git exits non-zero → return empty results
    - If not a git repo → return empty
    - If no remote named origin → return empty

    Follow Neovim Lua conventions: `local M = {}`, 2-space indent, pcall for unsafe ops.
  </action>

  <done>
