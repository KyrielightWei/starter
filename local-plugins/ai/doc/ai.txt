==============================================================================
CONTENTS                                                        *ai-contents*

    1. Commands ............... |ai-commands|
    2. Keymaps ................ |ai-keymaps|
    3. Configuration .......... |ai-configuration|
    4. Modules ................ |ai-modules|

==============================================================================
1. COMMANDS                                                        *ai-commands*

High-frequency~

AISync              Select and sync config to a tool (OpenCode/Claude Code/Pi)
AIKeys              Edit API keys and base URLs
AIProvider          Open Provider Manager panel
AIModel             Switch model (with scope: global/tool-specific)

Tool config~

OpenCodeGenerate    Generate OpenCode config from template
OpenCodePreview     Preview merged OpenCode config
OpenCodeEdit        Edit OpenCode template
OpenCodeStatus      Show OpenCode installation and config status
OpenCodeTheme       TUI theme: `OpenCodeTheme [generate|preview|edit]`

ClaudeCodeGenerate  Generate Claude Code settings
ClaudeCodePreview   Preview Claude Code settings
ClaudeCodeEdit      Edit Claude Code template
ClaudeCodeStatus    Show status including dependency check

PiGenerate          Generate Pi config
PiPreview           Preview Pi config
PiEdit              Edit Pi settings template
PiStatus            Show Pi CLI and config status

AI Review~

AIReviewStart       Start an AI review session
AIReviewAdd         Add review comment at current cursor location
AIReviewPanel       Open review comment panel
AIReviewExport      Export notes.md, notes.json, and fix-prompt.md
AIReviewStatus      Show active review session status
AIReviewClose       Close active review session

Subcommands~

AI template list [tool]           List template versions
AI template select [tool]         Select template version (picker UI)
AI template create <tool> <name>  Create new template version
AI template delete <tool> <name>  Delete template version
AI template rename <tool> <o> <n> Rename template version
AI template edit                  Edit current template version

AI context copy                   Copy buffer context to clipboard
AI context show                   Show context in new buffer

AI prompt edit                    Open prompts directory
AI prompt list                    List available prompts

AI watch                          Enable config file watcher
AI watch force                    Force sync all configs

AI export                         Export API keys to env file

AI backup <tool> [n]              Restore config backup

==============================================================================
2. KEYMAPS                                                        *ai-keymaps*

All keymaps use the `<leader>k` prefix (AI Tools group):

<leader>kk    AIModel         Switch model
<leader>ks    AISync          Sync config
<leader>ke    AIKeys          Edit API keys
<leader>kp    AIProvider      Provider Manager

<leader>kC    Commit Picker   Open commit picker
<leader>kf    Next Commit     Navigate to next commit
<leader>kb    Prev Commit     Navigate to previous commit
<leader>kd    Diff Viewer     Open diff viewer

<leader>krr   Review Start    Start AI review session
<leader>kra   Review Add      Add review comment
<leader>krl   Review Panel    Open review panel
<leader>krx   Review Export   Export review artifacts
<leader>krs   Review Status   Show review session status

==============================================================================
3. CONFIGURATION                                            *ai-configuration*

require("ai").setup({opts})

Options:~
    template_dir  (string)  Base directory for template files.
                            Default: `vim.fn.stdpath("config")`

Template files are read from:
    {template_dir}/templates/{tool}/{version}.template.jsonc
    {template_dir}/{tool}.template.jsonc  (legacy fallback)
    {template_dir}/pi/                    (Pi resource templates)
